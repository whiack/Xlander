1. Environment Requirements
- X11 is required to install first to run the program. 

2. Description
This Xlander program is a simple design of a Lander game. User is able to control the spaceship using keyboard. 

Arrow Up: Move up
Arrow Down: Move down
Arrow Left: Move left
Arrow Right: Move right
Space: pause / restart /start
q: quit 

User needs to land the spaceship on one of the pads. 

3. Compile the program
Make

4. Run the program
Make run
